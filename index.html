GitHub Pages 部署指南

  1. 创建 GitHub 账号（如果还没有）
  2. 新建一个仓库，命名为 torties-redirect
  3. 上传此 HTML 文件，命名为 index.html
  4. 进入仓库设置 → Pages
  5. 选择分支（通常是 main）
  6. 选择根目录 / (root)
  7. 保存设置，等待部署完成
  8. 您的链接：https://用户名.github.io/torties-redirect
